Prerequisites

Before installing RatAllOfYou, make sure you have the following:

  • Minecraft Launcher
  • Forge 1.8.9
  • Java 8 or higher
  • Windows 10/11, macOS, or Linux
1

Install Minecraft Launcher and RatAllOfYou

Download and install the official Minecraft Launcher from the Minecraft website. Make sure you have purchased a Minecraft account.

2

Download latest 1.8.9 Forge mod and install for client

Download the Forge 1.8.9 installer from the Forge website. Run the installer and select "Install client" option.

3

Run Minecraft 1.8.9 once

Open the Minecraft Launcher, select the Forge 1.8.9 profile, and click "Play". This will create the necessary files and folders for Forge.

4

Open the .minecraft folder

Press Win+R and type "%appdata%" (without quotes) in the Run dialog box. Press Enter to open the Roaming folder. Find and open the .minecraft folder.

5

Find .minecraft/mod folder

Inside the .minecraft folder, look for the "mods" folder. If it doesn't exist, create a new folder and name it "mods".

6

Install RatAllOfYou

Download the RatAllOfYou.jar file from the GitHub releases page and move it into the "mods" folder. Make sure you're using the correct version for your Forge installation.

Troubleshooting

Having issues with the installation? Check out these common problems and solutions:

Minecraft crashes when launching with the mod

Make sure you're using the correct version of Forge for the mod. RatAllOfYou requires Forge 1.8.9. Also, check if you have any conflicting mods installed. Try removing other mods to see if the issue persists.

The mod doesn't appear in the mods list

Ensure that the mod file is placed in the correct "mods" folder. The folder should be located in %appdata%/.minecraft/mods. Also, verify that you're using the correct version of the mod for your Minecraft version.

Some features aren't working correctly

Check if you're using the latest version of the mod. You can download updates from the GitHub releases page. If the problem persists, report it on the GitHub issues page with detailed information about the issue.

How do I update the mod?

To update the mod, simply download the latest version from GitHub and replace the old .jar file in your mods folder with the new one. Make sure to back up your old version in case you encounter any issues with the update.
